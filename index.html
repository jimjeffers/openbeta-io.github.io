Why Another Climbing Website?

Climbing pioneers have created a gift for generations of climbers to enjoy.

Today's crowdsourcing climbing sites while receiving and benefiting from contents curated by their user base have closed off access to such data.

By opening up and sharing climbing data we continue to uphold the knowledge sharing ethos.

Let's Make A Difference

Open Data

Community created climbing betas and related contents are available for download via an API. Public redistribution of data is encouraged as long as their modifications and source code stay open.

Open Source

Community participation does not stop at climbing beta. You can write and test code to help improve the site. Contents and source code are licensed under Creative Commons (BY-SA) and GNU GPL.

A Fresh Start

In the age of social media, GPS-enabled smartphones, advanced GIS and web technologies we believe a modern climbing site is long overdue.
